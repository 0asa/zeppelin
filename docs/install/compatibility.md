Currently, Zeppelin distribution includes Hive 0.11 jdbc driver implementation. Following system considered to compatible.

 * Hive 0.11 (with hiveserver2)
 * Hive 0.9  (with hiveserver)
 * Shark 0.8 (with sharkserver)
